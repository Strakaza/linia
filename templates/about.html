À propos de Linia : Votre comparateur de bus

Linia est un outil de visualisation de données de transport public pour les lignes de bus longue distance.

Visualisation des réseaux

L'application agrège les données de deux opérateurs : Flixbus et BlaBlaCar Bus. La recherche permet d'afficher les lignes desservant un arrêt spécifique.

Planification par étapes

Le planificateur permet de construire un itinéraire séquentiel. Lors de l'ajout d'une ville, une liste de destinations directement connectées est générée pour permettre la sélection de l'étape suivante.

Exportation des données

Les itinéraires créés peuvent être exportés au format PDF. Ce document contient le rappel des étapes et une capture de la carte associée au trajet.

FAQ

Linia est-il gratuit ?
Oui, l'outil est entièrement gratuit et sans publicité.
Quels sont les pays couverts ?
Linia couvre principalement l'Europe : France, Allemagne, Espagne, Italie, Benelux, et d'autres pays européens desservis par FlixBus et BlaBlaCar Bus.
À quelle fréquence les données sont-elles mises à jour ?
Les données GTFS sont mises à jour le premier jour de chaque mois.
Comment signaler une erreur ou proposer une amélioration ?
Vous pouvez ouvrir une issue sur le dépôt GitHub du projet ou contacter le développeur via contact@liniabus.eu

Sources de données (GTFS)

Les données sont issues des flux GTFS officiels disponibles sur transport.data.gouv.fr. Linia effectue une unification de ces flux pour permettre une consultation centralisée.
